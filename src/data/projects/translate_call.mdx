# Translate-Call

## Overview
Translate-Call is an offline, real-time translation application for Android. It enables users to have fluid conversations in different languages by capturing audio, transcribing it, translating it, and synthesizing speechâ€”all processing happening directly on the device for privacy and speed.

## Features
- **Offline Translation**: Uses Meta's NLLB model for high-quality offline translation.
- **Speech Recognition**: Powered by OpenAI's Whisper for state-of-the-art speech-to-text.
- **Conversation Mode**: Connects two phones via Bluetooth/Wi-Fi for seamless bilingual conversation.
- **Walkie-Talkie Mode**: Single-device mode for quick interactions.
- **Privacy Focused**: All processing data stays on the device.

## Tech Stack
- **Languages**: Kotlin (Android), Python (Model processing)
- **AI Models**: NLLB-200, OpenAI Whisper
- **Architecture**: MVVM
