# DataQuery AI Assistant

## Overview
DataQuery AI Assistant simplifies data analysis by allowing users to interact with their datasets using plain English. It bridges the gap between raw data and actionable insights through an intuitive chat interface and dynamic visualizations.

## Key Features
- **Natural Language Queries**: Ask questions about your data in English.
- **Dynamic Visualizations**: Automatically generates relevant charts (Bar, Line, Pie) based on the query result.
- **Voice Input**: Support for voice commands to query data.
- **Data Export**: Export analysis results and charts for reporting.
